Advanced Data Analytics — Machine Learning for Analysts

🔹 Lesson 1: Introduction to Machine Learning for Data Analysts
================================================================
Summary:
Machine Learning (ML) helps analysts move from descriptive (“what happened”) to predictive (“what will happen”).
Supervised Learning: Predict known outcomes (e.g., sales forecast)
Unsupervised Learning: Discover patterns (e.g., customer segmentation)
Common algorithms: Linear Regression, Decision Trees, K-Means

Example:
A retail analyst predicts next month’s sales using historical sales + seasonality via regression.

Video: Machine Learning for Data Analysts – 15 mins


🔹 Lesson 2: Predictive Modeling (Linear Regression)
==========================================================
Summary:
Predictive modeling uses past data to estimate future outcomes.
Linear Regression: Relationship between variables
Equation: y = mx + c
Metrics: R² score, RMSE

Example:
Predict house prices based on area, location, and number of rooms.

Video: Linear Regression with Python – 20 mins


🔹 Lesson 3: Classification Models for Analysts
===================================================
Summary:
Classification = predicting categories (e.g., Yes/No, High/Low).
Algorithms: Logistic Regression, Decision Trees, Random Forest
Metrics: Accuracy, Precision, Recall, F1-score

Example:
A telecom analyst predicts whether a customer will churn (Yes/No).

Video: Classification Models Explained – 18 mins
